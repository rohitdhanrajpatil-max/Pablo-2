## 2025-05-14 - Optimizing Hospitality Feedback Forms
**Learning:** For hospitality-specific guest feedback forms, using `autoComplete="organization"` for hotel names and `autoComplete="tel"` for mobile contact fields significantly reduces friction on mobile devices by leveraging browser autofill.
**Action:** Always include specific `autoComplete` attributes and link labels via `id`/`htmlFor` in hospitality guest forms.
